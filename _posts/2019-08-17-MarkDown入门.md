MarkDown是一种标记语言，由于其易读易写，兼容HTML，常用于文档的排版。GitHub中的.md文件和issue都是用MarkDown来写的，许多博客网站也是基于MarkDown的(包括我的博客网站)。接下来，我将简要介绍一下MarkDown的常用语法：
# 1.段落和换行：
段落的前后要有一个以上的空行。
换行需要在结尾有两个及以上的空格。
# 2.标题：
使用#的数目来表示标题的级数。最多六个。注意#之后有空格。
```markdown
# 第一级标题
## 第二级标题
###### 第六级标题
```
# 第一级标题
## 第二级标题
###### 第六级标题
# 3.字体：
粗体：
```markdown
**粗体**
__粗体__
```
**粗体**  
__粗体__  

斜体：
```markdown
*斜体*
_斜体_ 
```
*斜体*  
_斜体_   

横线：
```markdown
~~横线~~
```
~~横线~~    

混用粗体和斜体：
最好外面的用_，里面的用*
```markdown
__粗体*粗斜体*粗体__
_斜体**粗斜体**斜体_
```
__粗体*粗斜体*粗体__  
_斜体**粗斜体**斜体_  
# 4.引用文本：
在一段的第一行或者所有行使用>。每一个> 之后都有空格。可以通过多个> 实现嵌套引用。
```markdown
他说  
> 她说 
>  
> > 它说
> >   
> > > 汪  
> > > 喵  
> >  
> > 它什么也不说  
>  
> 她什么也不说 
```
他说  
> 她说 
>  
> > 它说
> >   
> > > 汪  
> > > 喵  
> >  
> > 它什么也不说  
>  
> 她什么也不说  
# 5.引用代码：
使用配对的单个反引号会引用一小段代码  
\`code\`  
`code`  
使用配对的三个反引号可以引用一个区域的代码  
\```  
code  
code  
code  
\```  
```markdown
code
code
code
```
可以在第一个(```)添加编程语言来选择渲染方式
```cpp
#include<iostream>
#include<string>
using namespace std;
int main()
{
    string a = "a";
    cout << a << endl;
}
```
# 6.链接： 
先在中括号中写链接的文本，然后在小括号中写链接的URL。
```markdown
欢迎来到我的[博客](https://dayekuaipao.github.io/blog/)。
```
欢迎来到我的[博客](https://dayekuaipao.github.io/blog/)。
可以使用./和../等相对链接。
还可以使用参考文献式链接"
```markdown
I get 10 times more traffic from [Google] [1] than from
[Yahoo] [2] or [MSN] [3].

  [1]: http://google.com/        "Google"
  [2]: http://search.yahoo.com/  "Yahoo Search"
  [3]: http://search.msn.com/    "MSN Search"
```
I get 10 times more traffic from [Google] [1] than from
[Yahoo] [2] or [MSN] [3].
[1]: http://google.com/        "Google"
[2]: http://search.yahoo.com/  "Yahoo Search"
[3]: http://search.msn.com/    "MSN Search"
如果直接以链接地址作为链接文本，可以用如下快捷写法：
```markdown
<http://www.shengbin.me>
```
<http://www.shengbin.me>
# 7.图片：
类似于链接，前面加一个感叹号。
```markdown
![sakura fes](../SakuraFes.jpg)
```
![sakura fes](../SakuraFes.jpg)
# 8.列表：
使用-,+或者*加空格可以形成无序列表。
```markdown
- a
- b
- c
+ a
+ b
+ c
*
* a
* b
* c
```
- a
- b
- c
+ a
+ b
+ c
* a
* b
* c
使用缩进可以形成嵌套式列表：
```markdown
- a
    - b
    - c
- d
```
- a
    - b
    - c
- d
使用数字加句号加空格可以形成有序列表：
1. a
2. b
3. c
# 9.任务列表：
对于任务列表，使用- \[  \] ，对于已经完成的可以使用- \[x\] ：
```markdown
- [ ] 没做
- [ ] 没做
- [x] 做了
```
- [ ] 没做
- [ ] 没做
- [x] 做了  

此时，如果一个描述是以括号开始的，需要用/将其隔开。
# 10.分割线：
一行中用三个以上的*，-，_
```markdown
---
***
___
```
---
***
___
# 11.插入表格：
使用|和---可以插入表格：
```markdown
1|2  
---|---  
a|b  
c|d  
```
1|2  
---|---  
a|b  
c|d  
# 转义字符:
```markdown
\   backslash
`   backtick
*   asterisk
_   underscore
{}  curly braces
[]  square brackets
()  parentheses
#   hash mark
+   plus sign
-   minus sign (hyphen)
.   dot
!   exclamation mark
```
